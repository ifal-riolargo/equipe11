Sobre o Apache Wicket

O Apache Wicket é um framework orientado a componentes que contém um suporte eficaz e independente a componentes que seguem um modelo, fazendo com que novas instâncias sejam “plugadas” no framework com simplicidade. Trazendo de forma simples a separação de responsabilidades entre as classes Java e o HTML, onde o HTML é usado para criação da interface visual e o Java para o controle de estado e também modelo.

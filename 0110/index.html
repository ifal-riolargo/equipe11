1011-AW

O que é a 1011-AW?

Uma ferramenta de treinamento para usuários desenvolverem habilidades em Apache Wicket.

O que o nosso treinamento oferece?

Neste curso online os alunos vão aprender a trabalhar com desenvolvimento web utilizando o Apache Wicket, que permite com que os projetos sejam criados com mais velocidade e desempenho. Ao final do curso, espera-se que o aluno consiga criar uma aplicação completa, por mais simples que ela seja, utilizando a ferramenta.
